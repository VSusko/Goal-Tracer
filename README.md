# Goal Tracer

## Ideia geral do projeto:
A ideia do projeto é criar um sistema que permita ao usuário planejar sua rotina e ter um controle do tempo gasto em cada atividade. A principal utilidade do projeto é permitir que os usuários tomem ciência dos tempos gastos em cada atividade e consigam ajustá-las de acordo com suas metas. Dessa forma, espera-se que ao final do projeto: 
1) o usuário consiga registrar suas atividades para o dia ou semana
2) o usuário consiga marcar as atividades como feitas 
3) o usuário consiga depositar um quantidade de horas empreendidas em uma atividade 
4) o usuário consiga estabelecer metas para a semana (em horas ou atividades)
5) o sistema calcule automaticamente o somatório de todas as horas empreendidas em cada atividade da meta 
6) o sistema mostre se a meta para a semana foi atingida ou não

## Descrição geral do sistema:
O sistema desenvolvido é um portal de atividades. Cada usuário pode cadastrar, ler, deletar e atualizar diferentes atividades planejadas para o dia. Após o login, há uma tela com tarefas para fazer e as tarefas já feitas, juntamente com um painel de metas que o usuário estabelece
